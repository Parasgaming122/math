Forms of a Quadratic

Standard form$ax^2 + bx + c$
Factored form$a(x - \alpha)(x - \beta)$
Vertex / completed-square form$a(x + p)^2 + q$

Vertex is at $(-p, q)$. Minimum if $a > 0$, maximum if $a < 0$.

Completing the Square

Method

$x^2 + bx + c \;\longrightarrow\; \left(x + \frac{b}{2}\right)^2 - \frac{b^2}{4} + c$

For $ax^2 + bx + c$: factor out $a$ first, then complete square inside the bracket.

Example: $2x^2 - 8x + 5$

$= 2(x^2 - 4x) + 5$
$= 2[(x-2)^2 - 4] + 5$
$= 2(x-2)^2 - 8 + 5$
$= 2(x-2)^2 - 3$

Vertex: $(2, -3)$. Since $a = 2 > 0$, this is a minimum.

Common Mistake: Forgetting to multiply the constant back by $a$ after completing the square inside brackets. Always expand and simplify fully.

Quadratic Formula & Discriminant

Quadratic formula (for $ax^2+bx+c=0$)$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
Discriminant $\Delta$$\Delta = b^2 - 4ac$
$\Delta$Roots
$> 0$2 distinct real roots
$= 0$1 repeated root (tangent)
$< 0$No real roots
Key use: If a line is tangent to a curve, set them equal → $\Delta = 0$. If it intersects, $\Delta > 0$. If no intersection, $\Delta < 0$.

Sum & Product of Roots

$\alpha + \beta = -\frac{b}{a} \qquad \alpha\beta = \frac{c}{a}$

Intersection of Line and Curve

Example: Find where $y = x^2 + 2$ meets $y = 3x$

$x^2 + 2 = 3x$
$x^2 - 3x + 2 = 0$
$(x-1)(x-2) = 0$
$x = 1 \Rightarrow y = 3$, $x = 2 \Rightarrow y = 6$
Points: $(1,3)$ and $(2,6)$

Exam Tip: When the question says "find the values of $k$ for which the line is tangent to the curve", always set equal, rearrange to standard form, then set $\Delta = 0$.

Quadratic Inequalities

Solve $ax^2 + bx + c > 0$ (or $< 0$):

Steps

1. Factorise (or use formula) → find roots $\alpha, \beta$ ($\alpha < \beta$)
2. Sketch the parabola
3. $> 0$: outside region → $x < \alpha$ or $x > \beta$
4. $< 0$: inside region → $\alpha < x < \beta$

Example: $x^2 - 5x + 6 < 0$

$(x-2)(x-3) < 0$
Roots: 2, 3. Parabola opens up.
Answer: $2 < x < 3$

Common Mistake: Dividing both sides of an inequality by a negative number without flipping the sign. Also, never divide by $x$ (it could be negative).

Quadratic in Disguise

$x^4 - 5x^2 + 4 = 0$
Let $u = x^2$: $u^2 - 5u + 4 = 0$
$(u-1)(u-4) = 0 \Rightarrow u = 1$ or $4$
$x = \pm 1, \pm 2$

Simultaneous Equations (Linear + Quadratic)

Method

1. From linear equation, express one variable in terms of the other
2. Substitute into quadratic equation
3. Solve the resulting quadratic
4. Substitute back to find the other variable

Definitions

DomainSet of allowed $x$-values (inputs)
Range (codomain)Set of resulting $y$-values (outputs)
FunctionEach $x$ maps to exactly ONE $y$
TypeMeaningHas Inverse?
One-to-oneEach $y$ comes from 1 $x$Yes
Many-to-oneMultiple $x$ give same $y$No

Restrict domain to make many-to-one → one-to-one, then inverse exists.

Example: Restricting Domain

$f(x) = x^2$ is many-to-one on $\mathbb{R}$.
Restrict to $x \geq 0$: now one-to-one, inverse is $f^{-1}(x) = \sqrt{x}$.
Restrict to $x \leq 0$: now one-to-one, inverse is $f^{-1}(x) = -\sqrt{x}$.

Exam Tip: "Find the greatest/least value of $a$ such that $f$ has an inverse" → find the vertex and restrict at that point.

Composite Functions

$fg(x)$ means$f$ applied to $g(x)$: $f(g(x))$
Order matters! $fg \neq gf$ in general. Apply right to left: $fg$ → do $g$ first, then $f$.
Example: $f(x)=2x+1$, $g(x)=x^2$

$fg(x) = f(x^2) = 2x^2 + 1$
$gf(x) = g(2x+1) = (2x+1)^2 = 4x^2+4x+1$

Domain of $fg$$x$-values in domain of $g$ such that $g(x)$ is in domain of $f$

Solving Composite Function Equations

Example: Solve $fg(x) = 7$

$2x^2 + 1 = 7$
$2x^2 = 6 \Rightarrow x^2 = 3$
$x = \pm\sqrt{3}$

Check solutions are in the domain of $g$!

Inverse Functions

Relationship$ff^{-1}(x) = f^{-1}f(x) = x$
Domain of $f^{-1}$$=$ Range of $f$
Range of $f^{-1}$$=$ Domain of $f$

Finding $f^{-1}(x)$

Steps

1. Write $y = f(x)$
2. Rearrange to make $x$ the subject
3. Replace $x$ with $f^{-1}(x)$ and $y$ with $x$

Example: $f(x) = \frac{2x+3}{x-1}$, $x \neq 1$

$y = \frac{2x+3}{x-1}$
$y(x-1) = 2x+3$
$xy - y = 2x + 3$
$xy - 2x = y + 3$
$x(y-2) = y+3$
$x = \frac{y+3}{y-2}$
$f^{-1}(x) = \frac{x+3}{x-2}$, $x \neq 2$

Geometrically: $y = f^{-1}(x)$ is the reflection of $y = f(x)$ in the line $y = x$.
Common Mistake: Forgetting to state the domain of the inverse function. The domain of $f^{-1}$ is the range of $f$.

Modulus Function $|x|$

$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$

Graph of $y = |f(x)|$: reflect any part below $x$-axis up.

Graph of $y = f(|x|)$: keep $x \geq 0$ part; reflect it in $y$-axis.

Solving $|f(x)| = k$

$|2x - 3| = 5$
Case 1: $2x-3 = 5 \Rightarrow x = 4$
Case 2: $2x-3 = -5 \Rightarrow x = -1$

Solving $|f(x)| = |g(x)|$

Method: Square both sides

$|2x-1| = |x+3|$
$(2x-1)^2 = (x+3)^2$
$4x^2 - 4x + 1 = x^2 + 6x + 9$
$3x^2 - 10x - 8 = 0$
$(3x+2)(x-4) = 0$
$x = -\frac{2}{3}$ or $x = 4$

Solving $|f(x)| < k$

$|f(x)| < k \Rightarrow -k < f(x) < k$, solve as compound inequality.

Common Mistake: In $|f(x)| > k$: this gives $f(x) > k$ OR $f(x) < -k$ (not AND!).

Converting to standard form

Complete the square on $x$ and $y$

$x^2+y^2-4x+6y-3=0$
$(x-2)^2-4 + (y+3)^2-9 - 3 = 0$

Centre $(2,-3)$, $r = 4$

Tangent to circle at $(x_1,y_1)$

Gradient of radius $= \frac{y_1-b}{x_1-a}$; tangent gradient $= -\frac{x_1-a}{y_1-b}$
Key property: Tangent $\perp$ radius at point of contact.
Exam Tip: If a question gives you a tangent and asks for the centre, use the fact that the line from centre to the point of tangency is perpendicular to the tangent.

Perpendicular Bisector & Circle from 3 Points

Perpendicular bisector of $AB$ passes through midpoint of $AB$ and has gradient $-\frac{1}{m_{AB}}$.

Finding circle equation from 3 points

1. Find perpendicular bisectors of two chords
2. Their intersection is the centre
3. Find radius = distance from centre to any point

Point inside/on/outside circle?

$d =$ distance from point to centre
$d < r \Rightarrow$ inside, $d = r \Rightarrow$ on, $d > r \Rightarrow$ outside

Line Intersecting Circle

Foot of perpendicular from point to line

Method

1. Find the gradient of the given line
2. Perpendicular has gradient $= -\frac{1}{m}$
3. Write equation of perpendicular through the external point
4. Solve simultaneously to find intersection (the foot)

Solving Trig Equations

Always find all solutions in the given range using CAST / symmetry / graph.

Steps for $\sin(n\theta + \alpha) = k$

1. Adjust the range: if solving for $0° \leq \theta \leq 360°$, new range for $(n\theta + \alpha)$ is $\alpha \leq n\theta+\alpha \leq 360°n + \alpha$
2. Find principal value: $\sin^{-1}(k)$
3. Use CAST to get ALL values of $(n\theta+\alpha)$ in the new range
4. Solve each for $\theta$

Example: solve $2\sin(x+30°)=1$, $0° \leq x \leq 360°$

$\sin(x+30°) = 0.5$
Range: $30° \leq x+30° \leq 390°$
$x+30° = 30°, 150°, 390°$ (next cycle)
$x = 0°, 120°, 360°$

Binomial Expansion

$(a + b)^n$, $n \in \mathbb{Z}^+$$(a+b)^n = \sum_{r=0}^{n} \binom{n}{r} a^{n-r} b^r$
Binomial coefficient$\binom{n}{r} = \frac{n!}{r!(n-r)!}$
General term ($(r+1)$th term)$T_{r+1} = \binom{n}{r} \cdot a^{n-r} \cdot b^r$
Expand $(2x + 3)^4$

$T_1 = \binom{4}{0}(2x)^4(3)^0 = 16x^4$
$T_2 = \binom{4}{1}(2x)^3(3)^1 = 4 \cdot 8x^3 \cdot 3 = 96x^3$
$T_3 = \binom{4}{2}(2x)^2(3)^2 = 6 \cdot 4x^2 \cdot 9 = 216x^2$
$T_4 = \binom{4}{3}(2x)^1(3)^3 = 4 \cdot 2x \cdot 27 = 216x$
$T_5 = \binom{4}{4}(2x)^0(3)^4 = 81$

Finding Specific Terms

Term Independent of $x$ (Constant Term)

Method

1. Write the general term $T_{r+1}$
2. Collect all powers of $x$
3. Set the total power of $x = 0$
4. Solve for $r$ (must be non-negative integer)

Example: Constant term in $\left(2x + \frac{3}{x}\right)^6$

$T_{r+1} = \binom{6}{r}(2x)^{6-r}\left(\frac{3}{x}\right)^r$
$= \binom{6}{r} \cdot 2^{6-r} \cdot 3^r \cdot x^{6-r} \cdot x^{-r}$
$= \binom{6}{r} \cdot 2^{6-r} \cdot 3^r \cdot x^{6-2r}$
Set $6 - 2r = 0 \Rightarrow r = 3$
$T_4 = \binom{6}{3} \cdot 2^3 \cdot 3^3 = 20 \cdot 8 \cdot 27 = 4320$

Finding a Specific Coefficient

Example: Coefficient of $x^3$ in $(1 + 2x)^8$

$T_{r+1} = \binom{8}{r}(1)^{8-r}(2x)^r = \binom{8}{r} \cdot 2^r \cdot x^r$
For $x^3$: $r = 3$
Coefficient $= \binom{8}{3} \cdot 2^3 = 56 \cdot 8 = 448$

Method: Write general term, collect powers of $x$, set power = required value, solve for $r$. If $r$ is not a non-negative integer $\leq n$, the term doesn't exist.

Arithmetic Progression (AP)

Sequence: $a, a+d, a+2d, \ldots$ (constant difference $d$)

$n$th term$u_n = a + (n-1)d$
Sum of first $n$ terms$S_n = \frac{n}{2}[2a + (n-1)d]$
Or equivalently$S_n = \frac{n}{2}(a + l)$ where $l$ = last term
Example: AP 3, 7, 11, ... Find $S_{20}$

$a=3$, $d=4$, $n=20$
$S_{20} = \frac{20}{2}[2(3) + 19(4)] = 10[6+76] = 10 \times 82 = 820$

Common Exam Setups

• "Sum of first $n$ terms $> 500$" → inequality in $n$, solve
• "Three consecutive AP terms" → let them be $a-d$, $a$, $a+d$
• "Four consecutive AP terms" → let $a-3d$, $a-d$, $a+d$, $a+3d$

$\sum_{r=1}^{n} r$$= \frac{n(n+1)}{2}$

Geometric Progression (GP)

Sequence: $a, ar, ar^2, \ldots$ (constant ratio $r$)

$n$th term$u_n = ar^{n-1}$
Sum of first $n$ terms$S_n = \frac{a(1-r^n)}{1-r}$ if $r \neq 1$
Sum to infinity ($|r| < 1$ only)$S_\infty = \frac{a}{1-r}$
$S_\infty$ exists only when $|r| < 1$ (i.e., $-1 < r < 1$)
Example: GP 4, 2, 1, 0.5, ... Find $S_\infty$

$a=4$, $r=0.5$
$S_\infty = \frac{4}{1-0.5} = \frac{4}{0.5} = 8$

Finding $r$ or $a$ from given info

Given $u_2=6$ and $u_5=48$:
$u_2 = ar = 6 \quad u_5 = ar^4 = 48$
Divide: $r^3 = 8 \Rightarrow r = 2$, $a = 3$

Three consecutive GP terms

Let them be $\frac{a}{r}$, $a$, $ar$ (product $= a^3$)
Exam Tip: "Show that the GP converges" means show $|r| < 1$. "Find the range of values of $k$ for which the series converges" → solve $|r(k)| < 1$.

Core Rules

Power rule$\frac{d}{dx}[x^n] = nx^{n-1}$

Chain rule$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$
$\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$
Chain rule: $y = (3x^2+1)^5$

Let $u = 3x^2+1$, $y = u^5$
$\frac{dy}{du} = 5u^4$, $\frac{du}{dx} = 6x$
$\frac{dy}{dx} = 5(3x^2+1)^4 \cdot 6x = 30x(3x^2+1)^4$

Product rule$\frac{d}{dx}[uv] = u'v + uv'$
Quotient rule$\frac{d}{dx}\left[\frac{u}{v}\right] = \frac{u'v - uv'}{v^2}$
Exam Tip: Before differentiating, always try to simplify first. E.g., $\frac{x^3+2x}{x} = x^2+2$ is much easier to differentiate.

$\frac{dV}{dt} = \frac{dV}{dr} \cdot \frac{dr}{dt} = 4\pi(25) \cdot 2 = 200\pi$ cm³/s

Optimisation Problems

Steps

1. Write quantity to optimise as function of ONE variable
2. Use constraints to eliminate extra variables
3. Differentiate; set $= 0$; solve
4. Verify it's max or min using $f''$ or sign change
5. Find the optimum value

From First Principles

$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

Core Rules

Power rule ($n \neq -1$)$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C$
Constant$\int k\,dx = kx + C$
Sum/difference$\int [f \pm g]\,dx = \int f\,dx \pm \int g\,dx$
Reverse chain (linear argument)$\int (ax+b)^n\,dx = \frac{(ax+b)^{n+1}}{a(n+1)} + C$
Example: $\int (3x+2)^4\,dx$

$= \frac{(3x+2)^5}{3 \times 5} + C = \frac{(3x+2)^5}{15} + C$

Always include $+C$ for indefinite integrals!
Common Mistake: Forgetting to divide by the coefficient of $x$ when integrating $(ax+b)^n$.

Standard Integrals

$f(x)$$\int f(x)\,dx$
$x^n$ ($n \neq -1$)$\frac{x^{n+1}}{n+1} + C$
$\frac{1}{x}$$\ln|x| + C$
$e^x$$e^x + C$
$e^{ax}$$\frac{1}{a}e^{ax} + C$
$\sin x$$-\cos x + C$
$\cos x$$\sin x + C$
$\sin(ax+b)$$-\frac{1}{a}\cos(ax+b) + C$
$\cos(ax+b)$$\frac{1}{a}\sin(ax+b) + C$
$\sec^2 x$$\tan x + C$
$\frac{1}{\sqrt{x}}$$2\sqrt{x} + C$
$\frac{1}{ax+b}$$\frac{1}{a}\ln|ax+b| + C$

Definite Integrals

Fundamental theorem$\int_a^b f(x)\,dx = [F(x)]_a^b = F(b) - F(a)$
No $+C$ for definite integrals (it cancels).
Example: $\int_1^3 (x^2 + 2x)\,dx$

$= \left[\frac{x^3}{3} + x^2\right]_1^3$
$= \left(\frac{27}{3} + 9\right) - \left(\frac{1}{3} + 1\right)$
$= 18 - \frac{4}{3} = \frac{50}{3}$

Properties

$\int_a^b f\,dx = -\int_b^a f\,dx$
$\int_a^a f\,dx = 0$
$\int_a^b f\,dx + \int_b^c f\,dx = \int_a^c f\,dx$

Area Under a Curve

Area between curve and $x$-axis$A = \left|\int_a^b y\,dx\right|$
If curve goes below $x$-axis: the integral gives a negative value. Split at $x$-intercepts and take absolute values of each part.
Example: area of $y=x(x-2)$ between $x=0$ and $x=2$

Curve below $x$-axis in $[0,2]$.
$\int_0^2 (x^2-2x)\,dx = \left[\frac{x^3}{3} - x^2\right]_0^2 = \frac{8}{3} - 4 = -\frac{4}{3}$
Area $= \left|-\frac{4}{3}\right| = \frac{4}{3}$

Area between two curves ($f$ above $g$)$A = \int_a^b [f(x) - g(x)]\,dx$
Steps for area between curves

1. Find intersection points (solve $f = g$)
2. Determine which is on top in the interval
3. Integrate (upper $-$ lower)

Volume of Revolution & Reverse Differentiation

Integration as Reverse Differentiation

Example: $\frac{dy}{dx} = 3x^2 - 4$, passes through $(2, 5)$

$y = \int (3x^2-4)\,dx = x^3 - 4x + C$
At $(2,5)$: $5 = 8 - 8 + C \Rightarrow C = 5$
$y = x^3 - 4x + 5$

Volume of Revolution (about $x$-axis)

$V = \pi \int_a^b y^2\,dx$
About $y$-axis$V = \pi \int_c^d x^2\,dy$
Example: $y = \sqrt{x}$, from $x=0$ to $x=4$, about $x$-axis

$V = \pi\int_0^4 (\sqrt{x})^2\,dx = \pi\int_0^4 x\,dx$
$= \pi\left[\frac{x^2}{2}\right]_0^4 = \pi \cdot 8 = 8\pi$

Common Mistake: Forgetting the $\pi$ in the volume formula! Also, make sure to square $y$ (not just integrate $y$).

Trapezium Rule

With $n$ strips ($n+1$ ordinates)$\int_a^b y\,dx \approx \frac{h}{2}[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})]$
where $h = \frac{b-a}{n}$
Memory: $\frac{h}{2} \times [\text{first} + \text{last} + 2(\text{all middle values})]$
Example: Estimate $\int_1^3 \sqrt{x}\,dx$ with 4 strips

$h = \frac{3-1}{4} = 0.5$
$x$: 1, 1.5, 2, 2.5, 3
$y$: 1, 1.225, 1.414, 1.581, 1.732
$\approx \frac{0.5}{2}[1 + 1.732 + 2(1.225 + 1.414 + 1.581)]$
$= 0.25[2.732 + 2(4.220)]$
$= 0.25[2.732 + 8.440] = 0.25 \times 11.172 = 2.793$

Accuracy: More strips = better estimate. Overestimates for concave up ($\cup$), underestimates for concave down ($\cap$).
Exam Tip: Set up a table of $x$ and $y$ values first, then apply the formula carefully. Don't rush this question — it's easy marks if you're systematic.
